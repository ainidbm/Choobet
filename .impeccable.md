## Design Context

### Users
- 奶茶爱好者，有"选择困难症"，不知道今天喝什么
- 主要使用场景：站在奶茶店门口 / 外卖 App 前犹豫不决，打开手机随手扭一下
- 移动端为主（边走边用），偶尔桌面浏览

### Brand Personality
- **可爱治愈、温暖日常、轻松不严肃**
- 扭蛋机隐喻：转动旋钮 → 胶囊掉落 → 打开的惊喜
- 情绪目标：期待感 → 小小的惊喜 → 满足的微笑
- 3 个词概括：治愈、俏皮、温暖

### Aesthetic Direction
- **Tone**: 可爱治愈 — 柔软圆润、粉彩色调、扭蛋机隐喻
- **参考**: 日本扭蛋机、Sonny Angel 包装、无印良品零食区
- **反例**: 花里胡哨的渐变、霓虹灯牌、赌场老虎机、Material Design 卡片套卡片
- **主题**: 暖色奶油底 + 柔和配色，light mode only
- **字体**: 圆体/卡通体做展示标题，系统无衬线做正文

### Design Principles
1. **仪式感优先** — 扭的过程比结果展示更重要，盲转 + 揭晓 = 惊喜
2. **柔软不刺眼** — 圆角、柔光、暖色调，没有任何锐利或饱和的元素
3. **不花哨** — 动画有意图，装饰服务于氛围，风格统一克制
4. **移动端核心** — 480px 容器，单手操作友好，所有按钮拇指可达
5. **独立自治** — 每个板块是自包含的小世界，自己的规则自己的数据
