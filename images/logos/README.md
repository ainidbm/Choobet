# Logo 图片

请将各家奶茶店的 logo 图片放入此目录。

## 文件要求

- 格式：PNG（推荐）或 JPG
- 建议尺寸：200×200 像素
- 透明背景为佳
- 文件名与 `data/shops.js` 中 `logo` 字段保持一致

## 需要的文件

| 文件名 | 对应店铺 |
|--------|----------|
| `mixuebingcheng.png` | 蜜雪冰城 |
| `chabaidao.png` | 茶百道 |
| `xicha.png` | 喜茶 |
| `guming.png` | 古茗 |
| `yidiandian.png` | 一点点 |

## 缺少 Logo 时的降级

如果某个 logo 文件不存在，应用中会显示店铺名称文字作为替代。
